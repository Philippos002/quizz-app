Quiz Application
A real-time quiz application built with Angular, Ionic, and Firebase. Users can test their knowledge with timed quizzes, compete on a global leaderboard, and track their progress.

    Features

    User Authentication
  
    Register: Create new account with username, email, and password validation
    Login: Secure authentication using Firebase Authentication
    Logout: Safely disconnect from the application
    Role-based Access: Admin and regular user roles for content management

    Quiz System
    
    Real-time Quiz: Answer questions with real-time validation
    60-Second Timer: Timed quiz experience with automatic submission
    Position-based Answers: Answers populate in their correct positions from database
    Instant Feedback: Immediate validation on Enter key press
    Score Tracking: Cumulative scoring system tracking total points
  
    Admin Panel
    
    Manage Questions: Add, edit, and delete quiz questions
    Question Management: Full CRUD operations for question database
    Modal Interface: User-friendly modals for question management
    Admin-only Access: Protected routes with role-based guards
  
    Leaderboard
    
    Global Rankings: View all users ranked by total score
    Real-time Updates: Leaderboard updates immediately after quiz completion
    Player Statistics: See username and accumulated points

  User Profile
  
  Personal Statistics: View total score, number of quizzes played, and best score
  Score History: Track your quiz performance over time
  Profile Management: Central hub for user information
